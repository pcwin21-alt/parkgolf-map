# 파크골프장 데이터 수집 스키마

## 권장 구글시트 컬럼

| 컬럼명 | 설명 | 예시 |
| --- | --- | --- |
| `id` | 사이트 내부 고유 ID | `gj_01` |
| `name` | 구장명 | `서봉파크골프장` |
| `region` | 시도 | `광주광역시` |
| `district` | 시군구 | `광산구` |
| `address` | 주소 | `광주광역시 광산구 서봉동 205-6` |
| `phone` | 대표 연락처 | `062-942-9927` |
| `status` | 화면 배지용 상태 | `운영중`, `확인필요` |
| `source` | 정보 출처 | `광주시 연락처 시트` |
| `updatedAt` | 최종 확인일 | `2026-04-01` |
| `holes` | 홀 수 | `36` |
| `hours` | 운영시간 | `09:00-18:00` |
| `closedDays` | 휴장일 | `월요일` |
| `fee` | 이용요금 | `무료` |
| `parking` | 주차 정보 | `가능`, `확인필요` |
| `restroom` | 화장실 정보 | `있음`, `확인필요` |
| `beginnerFriendly` | 초보자 추천 여부 | `추천`, `보통`, `확인필요` |
| `lat` | 위도 | `35.1506825427151` |
| `lng` | 경도 | `126.758268593881` |

## 수집 우선순위

1. 웹/공공 정보에서 후보값 수집
2. 전화 확인으로 `운영시간`, `휴장일`, `요금`, `주차`, `화장실` 확정
3. 현장 제보나 운영자 제보로 보완

## 입력 기준

- `status`
  - `운영중`: 연락처 또는 운영 정보가 확보된 경우
  - `확인필요`: 연락처/운영 정보 보강이 필요한 경우
- `source`
  - 출처명을 짧고 고정적으로 입력
  - 예: `광주시 연락처 시트`, `지자체 체육시설 안내`, `전화 확인`, `현장 제보`
- `updatedAt`
  - 반드시 `YYYY-MM-DD` 형식 사용
- `parking`, `restroom`, `beginnerFriendly`
  - 자유서술보다 `가능/불가/확인필요`, `있음/없음/확인필요`, `추천/보통/확인필요`처럼 짧게 입력

## 운영 원칙

- 사이트에는 `최종 업데이트`, `출처`, `상태 배지`를 함께 노출
- 방문 결정이 중요한 연령층 특성을 고려해 `전화` 정보를 최우선으로 관리
- 정보가 불확실하면 빈칸보다 `확인필요`로 명시
